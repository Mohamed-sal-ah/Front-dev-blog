# Front-dev Blog

This is a simple static blog app built with Next.js, Typescript and Markdown.

## Description

This blog uses markdown files to return frontmatter data and HTML content using gray-matter.

## Installation

To install the app use

```bash
yarn install
```

or

```bash
npm install
```

## Useage

To run the app use

```bash
npm run dev
```

or

```
yarn dev
```
